# Trigga — Landing Page

Landing page de Trigga, agencia de automatizaciones para negocios.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deploy en Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

Abrí [http://localhost:3000](http://localhost:3000) en el navegador.

## Deploy

Conectar el repo en [vercel.com](https://vercel.com) — detección automática de Next.js.
